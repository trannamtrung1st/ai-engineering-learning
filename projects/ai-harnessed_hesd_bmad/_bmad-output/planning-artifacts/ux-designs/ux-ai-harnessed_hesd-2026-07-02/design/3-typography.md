# Typography

- **Display / headings:** Archivo Black (`{typography.display}` desktop, `{typography.display-mobile}` on phone and QR Display session title).
- **Body / labels:** Space Grotesk (`{typography.body}`, `{typography.label}`).
- **QR countdown:** `{typography.qr-countdown}` — only on QR Display rotation timer.

## Hierarchy rules

- Student check-in outcome screens use `{typography.display-mobile}` for the outcome headline (Vietnamese).
- Instructor dashboard session title uses `{typography.h3}`.
- Table headers use `{typography.label}` uppercase tracking optional — never smaller than 12px.
- Vietnamese copy follows the same scale; line-height never below 1.4 on mobile outcome text.
