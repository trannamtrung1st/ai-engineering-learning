# Layout & Spacing

Base unit `{spacing.1}` (8px). Container max `{spacing.container-max}` with `{spacing.container-padding}` horizontal padding.

| Context | Layout rule |
|---|---|
| Student mobile | Single column, full-width CTAs, `{spacing.3}` vertical section gaps |
| Admin / Instructor web | Two-column max for forms; tables full width within container |
| QR Display | Centered QR frame, minimum 60% viewport width on projector |

Section rhythm: `{spacing.4}` between major page blocks; `{spacing.2}` between form fields.
