# Do's and Don'ts

| Do | Don't |
|---|---|
| 2px black borders on every elevated surface | Soft cards, material-style ripples |
| Brand yellow for one primary CTA per screen | Multiple competing yellow buttons |
| Status colors map 1:1 to attendance states | Color-only status without text label |
| Large type on QR Display for back-of-room legibility | Thin/light fonts on projected view |
| Vietnamese outcome copy in Space Grotesk at ≥16px | English error strings on student flow |
| Step-up shadow on hover for interactive cards | Blur shadows or gradient backgrounds |
