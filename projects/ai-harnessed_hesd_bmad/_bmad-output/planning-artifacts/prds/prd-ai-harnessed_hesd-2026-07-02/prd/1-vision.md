# 1. Vision

HESD workshops run with 100–150 students per cohort. Today, roll call and paper sign-in waste instructor time, produce error-prone records, and make proxy attendance easy.

This product is a **mobile-web attendance system** for workshop check-in: instructors activate a session with a rotating QR code; students scan and check in from their phones; attendance updates in realtime; exceptions are handled manually; organizers export CSV after each session. The system uses layered validation (login, short-lived QR, roster membership, one check-in per student, GPS geofence) to reduce casual fraud without claiming absolute anti-fraud.

MVP is an **internal tool** for the HESD program — not a school-wide attendance platform. Admin and Instructor are distinct roles with separate web surfaces; students use mobile web only.
