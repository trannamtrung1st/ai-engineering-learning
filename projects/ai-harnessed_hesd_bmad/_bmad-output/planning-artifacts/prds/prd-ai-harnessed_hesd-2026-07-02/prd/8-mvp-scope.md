# 8. MVP Scope

## 8.1 In Scope

- FR-1 through FR-18, NFR-1 through NFR-6
- Roles per spec `roles.md`
- QR and validation rules per `check-in-validation.md`
- Neobrutalism UI per project design system

## 8.2 Out of Scope for MVP

| Item | Reason |
|------|--------|
| SSO | Deferred to v2 if school IT requires |
| Password reset email flow | Admin re-provisions or sets password manually in pilot |
| Instructor mobile app | Web dashboard sufficient |
| Polygon geofences | Circular geofence adequate for classroom pilot |
| Real-time push notifications to students | In-room QR is the signal |
